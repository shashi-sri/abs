FreshPass is a program offered by Albertsons for all its banners that offer Grocery Delivery. FreshPass program allows online customers to pay a monthly or annual fee and join a program, by which they will be able to enjoy unlimited grocery delivery and benefits. Currently the FreshPass Program has 115k+ Subscribers. The FreshPass subscriber base has grown 409% since its launch in June 2021. FreshPass offers verious features such as:
1. Personalised Savings 2. Capability to target Audience with Discounted Plan / Extended Trial 3. Promo code capabilities (reusable & unique) 4. Wallet Integration - Pre-populate existing card.
