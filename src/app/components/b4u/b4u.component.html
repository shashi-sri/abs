Banners For you is a gateway application for all the offers from various sources like Stores, Manufacturers and from business. Banners are part of the Safeway For you Menu (on top of the Menu bar) which opens all the banner coupon information in terms of various modules called gallery, clip and myList.
There are 3 types banner offers: 1. Store Coupons : Store Coupons gets generated from stores near to the customer. 2. Manufacture Coupons : Manufacture coupons are allocated from third party Application called Neptune. 3. Personalised Coupons : Personalised coupons gets generated based on customer purchase history. So all these coupons can be redeemed at the time of checkout.
