ERUMS means Ecommerce Replatforming Using MicroServices. Earlier cart and checkout was handled by .NET monolithic APIs.To provide a more robust, optimized and Scalable service, the .Net API's was re-platformed to align with latest cloud based technology using java microservices.

Back End Services:

1) Cart Services - Create,Add/Update items to the cart [ Supports Albertson cart ,Market Place carts,Wine cart and MTO [ Made to Order ] carts.
2) Pricing Service - Service used internally to call EPE[promotion details], catalog details,Tax as well as Repricing.
3) Checkout Service - Service which is used for checkout .Captures user information like Address[UCA] , Slot Info[DPA ] and Payment[OSPG].
4) Order Service - Service which is used to place order and support Edit order. Order placed is send to order kafka which is being picked up down stream system for fulfillment.
5) Payment Service - Service used to call OSPG services.
6) 3p OrderService - A service used to create Replenish order[Subscribe and Save].
7) Promotion Services - A service used to handle promotion [ Moved to EPE ,no longer under development].
